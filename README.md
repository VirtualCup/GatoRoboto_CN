## 游戏介绍
![封面](https://github.com/VirtualCup/GatoRoboto_CN/blob/main/Preview/Cover.png?raw=true "封面")
《[神猫机兵](https://store.steampowered.com/app/916730)》是款黑白像素风格的类银河城动作冒险游戏，玩家将扮演一只可爱的小猫咪去探索隐藏在神秘研究所的秘密。

## 补丁目的
修正《神猫机兵》的官方简中翻译。

## 参与成员
全部：Virtualname   
字体：最像素 EX2

## 安装方法
1. 到【[Release](https://github.com/VirtualCup/GatoRoboto_CN/releases "发布页面")】界面下载【GatoRoboto.zip】文件
2. 解压后覆盖原来的游戏文件，注意对准文件路径。

## 补充说明
* 补丁对应操作系统为Windows，对应游戏平台为Steam。
* 补丁为完全汉化，支持Steam成就解锁。
* 补丁对应的游戏内容版本ID可在【游戏库 > 右键属性 > 本地文件】的左下方查看。
